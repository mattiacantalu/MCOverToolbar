# MCOverToolbar
Reproduced `UIToolbar` over `UITabbar` effect (like File, Photos, WhatsApp and more).

Animations implemented: 
1. fade in/out
2. move up/down
3. no animation.

## Requirements
• Xcode 11.3
• Swift 5
